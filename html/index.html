🌐 HTML 핵심 문법 완전 가이드

📚 학습 목표: 이 튜토리얼을 통해 HTML의 핵심 문법을 단계별로 익히고, 실제 웹페이지를 만들 수 있는 기초를 다집니다.

1단계: HTML 기본 구조 이해하기

1.1 HTML 문서의 기본 골격

모든 HTML 문서는 다음과 같은 기본 구조를 가집니다:

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <title>페이지 제목</title> </head> <body> <!-- 여기에 실제 내용이 들어갑니다 --> </body> </html>
각 요소의 역할:
  • <!DOCTYPE html>: HTML5 문서임을 선언
  • <html>: 문서의 루트 요소
  • <head>: 메타데이터와 문서 정보
  • <body>: 실제 화면에 표시되는 내용

2단계: 텍스트 요소 다루기

2.1 제목 태그 (Heading Tags)

HTML은 6단계의 제목 태그를 제공합니다:

<h1>가장 중요한 제목</h1> <h2>두 번째 중요한 제목</h2> <h3>세 번째 중요한 제목</h3> <h4>네 번째 중요한 제목</h4> <h5>다섯 번째 중요한 제목</h5> <h6>여섯 번째 중요한 제목</h6>

가장 중요한 제목

두 번째 중요한 제목

세 번째 중요한 제목

네 번째 중요한 제목

다섯 번째 중요한 제목
여섯 번째 중요한 제목

2.2 문단과 텍스트 서식

<p>이것은 일반적인 문단입니다.</p> <p><strong>굵은 글씨</strong>와 <em>기울임 글씨</em>를 사용할 수 있습니다.</p> <p><mark>하이라이트된 텍스트</mark>와 <del>삭제된 텍스트</del>도 가능합니다.</p> <p>줄바꿈을 하려면 <br> 태그를 사용합니다.</p> <p><small>작은 글씨</small>와 <sub>아래첨자</sub>, <sup>위첨자</sup>도 지원됩니다.</p>

이것은 일반적인 문단입니다.

굵은 글씨와 기울임 글씨를 사용할 수 있습니다.

하이라이트된 텍스트와 삭제된 텍스트도 가능합니다.

태그를 사용합니다.

3단계: 목록 만들기

3.1 순서가 있는 목록 (Ordered List)

<ol> <li>첫 번째 항목</li> <li>두 번째 항목</li> <li>세 번째 항목</li> </ol>
  1. 첫 번째 항목
  2. 두 번째 항목
  3. 세 번째 항목

3.2 순서가 없는 목록 (Unordered List)

<ul> <li>항목 1</li> <li>항목 2</li> <li>항목 3</li> </ul>
  • 항목 1
  • 항목 2
  • 항목 3

3.3 중첩 목록

<ul> <li>웹 개발 <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> </li> <li>백엔드 개발 <ol> <li>Node.js</li> <li>Python</li> <li>Java</li> </ol> </li> </ul>
  • 웹 개발
    • HTML
    • CSS
    • JavaScript
  • 백엔드 개발
    1. Node.js
    2. Python
    3. Java

8.1 비디오와 오디오

<!-- 비디오 --> <video width="400" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.ogg" type="video/ogg"> 브라우저가 비디오를 지원하지 않습니다. </video> <!-- 오디오 --> <audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> 브라우저가 오디오를 지원하지 않습니다. </audio>

10단계: 실습 프로젝트 - 간단한 개인 프로필 페이지

10.1 완성된 HTML 페이지 예시

지금까지 배운 모든 요소들을 활용하여 간단한 개인 프로필 페이지를 만들어보겠습니다:

<!DOCTYPE html> <html lang="ko"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>김개발의 프로필</title> </head> <body> <header> <h1>김개발 (Kim Developer)</h1> <nav> <a href="#about">소개</a> | <a href="#skills">기술</a> | <a href="#contact">연락처</a> </nav> </header> <main> <section id="about"> <h2>자기소개</h2> <p>안녕하세요! 저는 <strong>웹 개발자</strong>를 꿈꾸는 <em>김개발</em>입니다.</p> <p>현재 <mark>HTML, CSS, JavaScript</mark>를 공부하고 있습니다.</p> </section> <section id="skills"> <h2>기술 스택</h2> <ul> <li>HTML5</li> <li>CSS3</li> <li>JavaScript</li> <li>React (학습 중)</li> </ul> </section> <section id="contact"> <h2>연락처</h2> <p>이메일: <a href="mailto:kim@example.com">kim@example.com</a></p> <p>전화: <a href="tel:+821012345678">010-1234-5678</a></p> </section> </main> <footer> <p>© 2024 김개발. 모든 권리 보유.</p> </footer> </body> </html>
